SparsePolynomialZonotope

LazySets.SparsePolynomialZonotope вҖ” Type

Type that represents a sparse polynomial zonotope.

A sparse polynomial zonotope $\mathcal{PZ} вҠӮ \mathbb{R}^n$ is represented by the set

\[\mathcal{PZ} = \left\{x \in \mathbb{R}^n : x = c + вҲ‘бөўвӮҢвӮҒК°\left(вҲҸвӮ–вӮҢвӮҒбө– Оұ_k^{E_{k, i}} \right)Gбөў+вҲ‘вұјвӮҢвӮҒ^qОІвұјGIвұј,~~ Оұ_k, ОІвұј вҲҲ [-1, 1],~~ вҲҖ k = 1,вҖҰ,p, j=1,вҖҰ,q \right\},\]

where $c вҲҲ \mathbb{R}^n$ is the offset vector (or center), $Gбөў вҲҲ \mathbb{R}^{n}$ are the dependent generators, $GIвұј вҲҲ \mathbb{R}^{n}$ are the independent generators, and $E вҲҲ \mathbb{N}^{pГ—h}_{вүҘ0}$ is the exponent matrix with matrix elements $E_{k, i}$.

In the implementation, $Gбөў вҲҲ \mathbb{R}^n$ are arranged as columns of the dependent generator matrix $G вҲҲ \mathbb{R}^{n \times h}$, and similarly $GIвұј вҲҲ \mathbb{R}^{n}$ are arranged as columns of the independent generator matrix $GI вҲҲ \mathbb{R}^{nГ—q}$.

The shorthand notation $\mathcal{PZ} = \langle c, G, GI, E, idx \rangle$ is often used, where $idx вҲҲ \mathbb{N}^p$ is a list of non-repeated natural numbers storing a unique identifier for each dependent factor $ОұвӮ–$.

Fields

  • c вҖ“ offset vector
  • G вҖ“ dependent generator matrix
  • GI вҖ“ independent generator matrix
  • E вҖ“ exponent matrix
  • idx вҖ“ identifier vector of positive integers for the dependent parameters

Notes

Sparse polynomial zonotopes were introduced in [1].

  • [1] N. Kochdumper and M. Althoff. Sparse Polynomial Zonotopes: A Novel Set Representation for Reachability Analysis. Transactions on Automatic Control, 2021.

     [seed]::Union{Int, Nothing}=nothing)

Create a random sparse polynomial zonotope.

Input

  • SparsePolynomialZonotope вҖ“ type for dispatch
  • N вҖ“ (optional, default: Float64) numeric type
  • dim вҖ“ (optional, default: 2) dimension
  • nparams вҖ“ (optional, default: 2) number of parameters
  • maxdeg вҖ“ (optional, default: 3) maximum degree for each parameter
  • num_dependent_generators вҖ“ (optional, default: -1) number of dependent generators (see comment below)
  • num_independent_generators вҖ“ (optional, default: -1) number of independent generators (see comment below)
  • rng вҖ“ (optional, default: GLOBAL_RNG) random number generator
  • seed вҖ“ (optional, default: nothing) seed for reseeding

Output

A random sparse polynomial zonotope.

Algorithm

All numbers are normally distributed with mean 0 and standard deviation 1.

The number of generators can be controlled with the arguments num_dependent_generators and num_dependent_generators. For a negative value we choose a random number in the range dim:2*dim (except if dim == 1, in which case we only create a single generator). Note that the final number of generators may be lower if redundant monomials are generated.

LazySets.remove_redundant_generators вҖ” Method
remove_redundant_generators(S::SparsePolynomialZonotope)

Remove redundant generators from S.

Input

  • S вҖ“ sparse polynomial zonotope

Output

A new sparse polynomial zonotope where redundant generators have been removed.

Notes

The result uses dense arrays irrespective of the array type of S.

Algorithm

Let G be the dependent generator matrix, E the exponent matrix, and GI the independent generator matrix of S. We perform the following simplifications:

  • Remove zero columns in G and the corresponding columns in E.
  • Remove Zero columns in GI.
  • For zero columns in E, add the corresponding column in G to the center.
  • Group repeated columns in E together by summing the corresponding columns in G.
